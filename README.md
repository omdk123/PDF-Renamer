# PDF Renamer Automation Tool

A simple and efficient Python script designed to automate the process of renaming multiple PDF files in a directory. This tool helps in organizing file structures by batch-renaming files with a sequential naming convention.

## Features
- **Batch Processing:** Renames all PDF files in a specific folder at once.
- **Sequential Naming:** Automatically assigns names like `Notes_1.pdf`, `Notes_2.pdf`, etc.
- **Time Saving:** Replaces manual renaming with a single script execution.

## Requirements
- Python 3.x installed on your system.

## How to Use
1. Clone this repository or download the script file.
2. Place the script in the directory containing the PDF files you want to rename.
3. Open your terminal or command prompt in that directory.
4. Run the script using the following command:
   ```bash
   python rename.py
